- **01_basics**: تمرین‌های مقدماتی، متغیرها، ورودی، عملیات ریاضی و برنامه ساده Calculator

- **04_lists**: تمرین‌های لیست‌ها، افزودن/حذف عناصر، برش و List Comprehension
- **05_dicts**: تمرین‌های دیکشنری، لیست از دیکشنری‌ها و جمع‌بندی داده‌ها
